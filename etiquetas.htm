  • <base>

    • La etiqueta base define la URL base para todos los enlaces relativos de un documento. Debe tener presente un atributo href o un atributo target o ambos. Solo puede haber un único elemento base en un documento y debe estar dentro del elemento

    • BDI significa aislamiento bidireccional, se utiliza para asegurar la inclusion de texto que se lee de derecha a izquierda, en hebreo o arabe

  • <bdo>

    • BDO significa sobreescritura bidireccional, anula la dirección de lectura predeterminada del texto. Para leer el texto de derecha a izquierda o viceversa.
